快速開始

選擇您偏好的開發方式，幾分鐘內開始建構應用

Zephyr 環境設置與建構 
# 1. 一鍵設置 Zephyr 環境（首次執行）
./tools/zephyr/bootstrap.sh

# 2. 使用互動式選單（推薦）
./tools/zephyr/menu.sh

# 3. 或直接建構應用
./tools/zephyr/build.sh -a ipro_zephyr_demo -b ipro7_fpga -c

# 4. 手動使用 west 建構
cd zephyr-workspace
source setup_env.sh
west build -b ipro7_fpga ../apps/ipro_zephyr_demo
💡 提示：Zephyr 環境優化後僅需下載 6 個必要模組（約 125MB），建構時間約 30 秒。
FreeRTOS 建構指令 
# 使用簡化建構腳本
./build_freertos.sh ipro_ble_demo

# 或手動使用 CMake
cmake -S . -B build \
    -G "Ninja" \
    -DIPRO_SDK_BASE=$(pwd) \
    -DCONFIG_BUILD_PORJECT=ipro_ble_demo \
    -DCMAKE_PREFIX_PATH=$(pwd)/cmake \
    -DCUSTOM_CONFIG_DIR=$(pwd)/apps/ipro_ble_demo \
    -DCONFIG_APPS=1 \
    -DCROSS_COMPILE=/opt/toolchain/riscv_ipro7/bin/riscv64-unknown-elf-

cmake --build build
💡 提示：建構產出位於 apps/<PROJECT_NAME>/build/build_out/ 目錄。
