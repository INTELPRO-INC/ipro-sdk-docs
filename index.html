核心功能

IPRO SDK 提供完整的嵌入式開發生態系統，從底層驅動到高階應用一應俱全

AI NPU 加速

內建 0.1 TOPS NPU 神經網路加速器，支援邊緣運算、影像辨識、語音處理等 AI 應用場景。

影像處理

整合 ISP 影像處理管線、MJPEG 編碼、LCD 顯示驅動，支援攝影機串流與影像辨識應用。

語音處理

內建 AUADC/DAC 音訊子系統，支援 LC3 編解碼、LE Audio 串流，適用於語音辨識與音訊裝置開發。

Matter 智慧家庭

完整支援 Matter 智慧家庭協定，整合 BT/BLE、WiFi 6、Thread 多協定共存，實現跨平台互通。

BT/BLE 無線通訊

高效能 BLE 通訊（最高 999 Kbps），支援 LE Audio、多連線管理、OTA 韌體更新。

WiFi 6 與 Thread

WiFi 6 高速網路連線搭配 Thread Mesh 網路，內建 lwIP 協定棧與 Mbed TLS 安全加密。

模組化架構

清晰的分層架構設計：應用層、中間件、RTOS、BSP。各組件透過 Kconfig 系統靈活配置。

開發工具鏈

提供 make -C apps/<project> 快速建構、board_test.sh 自動化測試、serial_monitor.py 串口監控。

參考應用

GATT 服務 OTA 圖片傳輸
ipro_le_audio_headset LE Audio

LE Audio 耳機參考應用，基於 GAF/BAP 框架，支援 Unicast 雙向音訊串流、VCP 音量控制、MCP 媒體控制、USB 音訊介面。

GAF/BAP LC3 編解碼 Unicast USB 音訊
ipro_walkie_talkie 語音

LE Audio 對講機，支援 PTT 按鍵通話，BIS 廣播音訊雙向角色（Source/Sink），適用於群組通訊。

PTT BIS 廣播音訊
ipro_le_audio_bis_source

LE Audio 廣播發射端（Auracast TX），支援一對多音訊分發，PBP 廣播協定，USB 音訊輸入。

Auracast TX PBP USB 音訊
ipro_le_audio_bis_sink

LE Audio 廣播接收端（Auracast RX），自動同步廣播源，USB 音訊輸出。

Auracast RX 自動同步 USB 音訊
ipro7_pircam AI 影像

快速開始

選擇您偏好的開發方式，幾分鐘內開始建構應用

建構指令 
# 使用 Make 建構（推薦）
make -C apps/bluetooth/ipro_ble_remote

# 或使用建構腳本
./build_freertos.sh ipro_ble_remote build

# 或手動使用 CMake
cmake -S . -B build \
    -G "Ninja" \
    -DIPRO_SDK_BASE=$(pwd) \
    -DCONFIG_BUILD_PORJECT=ipro_ble_demo \
    -DCMAKE_PREFIX_PATH=$(pwd)/cmake \
    -DCUSTOM_CONFIG_DIR=$(pwd)/apps/ipro_ble_demo \
    -DCONFIG_APPS=1 \
    -DCROSS_COMPILE=/opt/toolchain/riscv_ipro7/bin/riscv64-unknown-elf-

cmake --build build
提示：建構產出位於 apps/<PROJECT_NAME>/build/build_out/ 目錄。
