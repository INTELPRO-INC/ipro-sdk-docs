IPRO SDK VS Code Extension

A VS Code extension designed specifically for IPRO SDK, supporting both local and remote development environments with one-click build and flash capabilities.

Installation Guide

Remote Linux Server Preparation

1. Install Required Tools

Install Development Tools 
# Install CMake and Ninja
sudo apt-get install cmake ninja-build

# Manual RISC-V toolchain installation (recommended)
# Download and extract toolchain to /opt/toolchain/riscv_ipro7/
echo 'export PATH=/opt/toolchain/riscv_ipro7/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

# Verify installation
which cmake ninja riscv64-unknown-elf-gcc
cmake --version
riscv64-unknown-elf-gcc --version

Local Windows Preparation

1. Install VS Code Extensions

  1. Install Visual Studio Code
  2. Install Remote - SSH extension
  3. Prepare flash tool ipro_iot_tool.exe

2. Flash Tool Path

The extension will automatically detect the following common paths:

Common Flash Tool Paths 
C:\Work\IPRO\ipro_sdk\tools\download\ipro_iot_tool.exe
C:\IPRO\tools\download\ipro_iot_tool.exe
C:\Program Files\IPRO\tools\download\ipro_iot_tool.exe

Install Extensions

  • IPRO SDK Extension: Install on remote server (via Remote SSH)

Installation Steps

  1. After connecting to the remote server, open Extensions panel (Ctrl+Shift+X)
  2. Search for IPRO SDK Extension, click "Install in SSH: server-name"
  3. Disconnect SSH or open a new local window
  4. Search for IPRO Flash Tool, click "Install" (make sure it's not "Install in SSH")
  5. Reload window: F1 â†’ Developer: Reload Window

Development Workflow

1. Connect to Remote Server

  1. Press F1 to open command palette
  2. Select Remote-SSH: Connect to Host...
  3. Select or add SSH host
  4. Enter password or use SSH key

2. Open SDK Folder

  1. Select File â†’ Open Folder...
  2. Select IPRO SDK root directory (e.g., /home/username/ipro_sdk/)

3. Use IPRO Sidebar

  1. Click the IPRO icon in the left activity bar
  2. Select project in Project Manager
  3. Click Configure button to configure
  4. Click Build button to compile

FAQ

Q1: Cannot find cmake or toolchain on remote

Problem: CMAKE_C_COMPILER not found in PATH

Solution:

  1. Verify tools are installed: which cmake riscv64-unknown-elf-gcc
  2. Add to PATH: echo 'export PATH=/opt/toolchain/riscv_ipro7/bin:$PATH' >> ~/.bashrc
  3. Reload window: F1 â†’ Developer: Reload Window

Q2: Local Flash Tool Extension not installed

Problem: Cannot call local Flash Tool Extension

Solution:

  1. Disconnect SSH, install in local VS Code
  2. Verify it shows "Install" not "Install in SSH"
  3. Reconnect to remote and reload window

Q3: Flash tool path not configured

Problem: Flash tool not configured

Solution:

  1. Run command: F1 â†’ IPRO: Configure Flash Tool Path
  2. Select ipro_iot_tool.exe location

Q4: COM port connection failed

Problem: shake hand failed

Solution:

  1. Verify USB cable is connected and driver is working
  2. Check if COM port number is correct
  3. Close other programs using the COM port
  4. Press RESET button and start flashing immediately

Q5: Firmware file does not exist

Problem: Firmware file does not exist, please build the project first

Solution:

  1. Verify Configure and Build have been executed
  2. Check apps/<project>/build/build_out/ directory
  3. If no files exist, re-run Build
