Full BLE 5.4 + LE Audio support

Low Power

Multiple power modes, sub-mA sleep current

Peripheral Connections

UART Debug Console

Default UART settings for debug console:

  • Baud Rate: 115200
  • Data Bits: 8
  • Stop Bits: 1
  • Parity: None
  • Flow Control: None

JTAG/SWD Debug Interface

Connect OpenOCD compatible debugger (J-Link, FTDI-based):

# OpenOCD connection
openocd -f tools/openocd/ipro7_fpga.cfg

# GDB connection
riscv64-unknown-elf-gdb -ex "target remote :3333"

RF Test Port

For conducted RF tests, connect test equipment to RF connector:

  • Connector Type: U.FL / IPEX
  • Impedance: 50Ω
  • Frequency Range: 2.4 GHz ISM band (2400-2483.5 MHz)

Power Supply Requirements

Parameter Min Typical Max Unit
Supply Voltage (VDD) 1.8 3.3 3.6 V

Important: Ensure stable power supply with adequate decoupling capacitors. Use 100nF ceramic capacitors close to power pins.
