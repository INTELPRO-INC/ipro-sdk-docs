IPRO7AI Chip Overview

AIOT 2.4G Multi-mode Wireless Chip

IPRO7AI is a multi-mode wireless chip designed for low-power AIOT applications, integrating Matter 1.4, BLE 5.4, Zigbee 3.0 and Thread 1.2, with a built-in AI NPU accelerator for smart home, image recognition, and IoT edge computing devices.

Matter 1.4 BLE 5.4 Zigbee 3.0 Thread 1.2 AI NPU 0.1 TOPS

MCU Processor

  • RISC-V RV32IMAFC @192MHz
  • L1 I/D Cache 16KB/16KB
  • RAM 300KB
  • ROM 128KB
  • eFuse 2Kb
  • RTC, Timer x2, WDT
  • DMA 8 Channels

Memory Configuration

  • Built-in 8Mb (1MB) Flash
  • Built-in 64Mb (8MB) pSRAM
  • XIP QSPI Support
  • On-The-Fly AES Decryption (OTFAD)
  • External QSPI Flash Support

Wireless Features

  • Matter 1.4 Full Support
  • Bluetooth 5.4 / BLE
  • Zigbee 3.0 / Thread 1.2
  • IEEE 802.15.4
  • Up to 16 BLE Connections
  • BLE/Zigbee/Thread Coexistence

Security Features

  • Secure Boot
  • Secure Debug
  • TrustZone Isolation
  • AES-CBC/CCM/GCM/XTS
  • MD5, SHA-1/224/256/384/512
  • TRNG (True Random Number Generator)
  • PKA (RSA/ECC Accelerator)

AI / NPU

  • NPU Performance: 0.1 TOPS
  • AI Detection Applications
  • Edge Computing Acceleration
  • Image/Voice AI Processing

Multimedia

  • Audio ADC 24-bit
  • Video MJPEG Codec
  • Camera DVP/SPI Interface
  • Display DBI/SPI Interface
  • USB 2.0

Design Recommendations

Input Protection

Add TVS (SMAJ15CA) for 12V input ESD/surge protection

Decoupling Capacitors

Place 100nF decoupling capacitor on each power pin, close to IC

Bulk Capacitors

Place 10μF~47μF capacitor on +5V and +3.3V rails

Power Sequencing

Ensure 0.9V core power establishes before IO power

Ground Planning

Use solid ground plane, separate analog/digital regions

LDO Selection

Use low-noise LDO for analog power (AVDD)
