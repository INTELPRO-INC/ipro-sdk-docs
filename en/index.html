Core Features

IPRO SDK provides a complete embedded development ecosystem, from low-level drivers to high-level applications

Matter Multi-Protocol

Full Matter 1.4 smart home protocol support with BLE 5.4 (up to 999 Kbps throughput), WiFi 6, Zigbee 3.0, Thread 1.2, and multi-protocol coexistence.

Dual RTOS Options

Support for both FreeRTOS and Zephyr RTOS. Zephyr offers modern development experience, Device Tree configuration, and fast build times (~30 seconds).

Modular Architecture

Clean layered architecture: Application, Middleware, RTOS, BSP. Components can be configured independently via Kconfig system.

Rich Component Library

Built-in lwIP network stack, Mbed TLS security, CherryUSB, multiple filesystem support for various application scenarios.

Development Toolchain

make -C apps/<project> quick build, Zephyr scripts, board_test.sh automated testing, serial_monitor.py serial monitoring for daily development.

AI NPU Acceleration

Built-in 0.1 TOPS NPU neural network accelerator for AI detection/recognition applications, edge computing, image recognition, and voice processing.

Complete CI/CD

Built-in GitLab CI/CD pipeline with automated build testing, memory analysis, security scanning for code quality assurance.

LE Audio Ready

Complete LC3 encoder/decoder support with Unicast and Broadcast Audio capabilities. Perfect for next-generation audio devices.

Quick Start

Choose your preferred development approach and start building in minutes

Zephyr Environment Setup & Build 
# 1. One-click Zephyr environment setup (first time)
./tools/zephyr/bootstrap.sh

# 2. Use interactive menu (recommended)
./tools/zephyr/menu.sh

# 3. Or build directly
./tools/zephyr/build.sh -a ipro_zephyr_demo -b ipro7_fpga -c

# 4. Manual west build
cd zephyr-workspace
source setup_env.sh
west build -b ipro7_fpga ../apps/ipro_zephyr_demo
Tip: Optimized Zephyr environment downloads only 6 essential modules (~125MB), build time ~30 seconds.
FreeRTOS Build Commands 
# Use Make build (recommended)
make -C apps/bluetooth/ipro_ble_remote

# Or use build script
./build_freertos.sh ipro_ble_remote build

# Or manual CMake build
cmake -S . -B build \
    -G "Ninja" \
    -DIPRO_SDK_BASE=$(pwd) \
    -DCONFIG_BUILD_PORJECT=ipro_ble_demo \
    -DCMAKE_PREFIX_PATH=$(pwd)/cmake \
    -DCUSTOM_CONFIG_DIR=$(pwd)/apps/ipro_ble_demo \
    -DCONFIG_APPS=1 \
    -DCROSS_COMPILE=/opt/toolchain/riscv_ipro7/bin/riscv64-unknown-elf-

cmake --build build
Tip: Build output is located in apps/<PROJECT_NAME>/build/build_out/ directory.
