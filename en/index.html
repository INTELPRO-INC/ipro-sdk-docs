Core Features

IPRO SDK provides a complete embedded development ecosystem, from low-level drivers to high-level applications

AI NPU Acceleration

Built-in 0.1 TOPS NPU neural network accelerator for edge computing, image recognition, and voice processing.

Image Processing

Integrated ISP image pipeline, MJPEG encoding, LCD display driver for camera streaming and image recognition applications.

Voice Processing

Built-in AUADC/DAC audio subsystem with LC3 codec support, LE Audio streaming for voice recognition and audio devices.

Matter Smart Home

Full Matter smart home protocol support with BT/BLE, WiFi 6, and Thread multi-protocol coexistence for cross-platform interoperability.

BT/BLE Wireless

High-performance BLE communication (up to 999 Kbps), LE Audio support, multi-link management, and OTA firmware updates.

WiFi 6 & Thread

WiFi 6 high-speed connectivity with Thread mesh networking, built-in lwIP stack and Mbed TLS security.

Modular Architecture

Clean layered architecture: Application, Middleware, RTOS, BSP. Components configured independently via Kconfig system.

Development Toolchain

make -C apps/<project> quick build, board_test.sh automated testing, serial_monitor.py serial monitoring.

Quick Start

Choose your preferred development approach and start building in minutes

Build Commands 
# Use Make build (recommended)
make -C apps/bluetooth/ipro_ble_remote

# Or use build script
./build_freertos.sh ipro_ble_remote build

# Or manual CMake build
cmake -S . -B build \
    -G "Ninja" \
    -DIPRO_SDK_BASE=$(pwd) \
    -DCONFIG_BUILD_PORJECT=ipro_ble_demo \
    -DCMAKE_PREFIX_PATH=$(pwd)/cmake \
    -DCUSTOM_CONFIG_DIR=$(pwd)/apps/ipro_ble_demo \
    -DCONFIG_APPS=1 \
    -DCROSS_COMPILE=/opt/toolchain/riscv_ipro7/bin/riscv64-unknown-elf-

cmake --build build
Tip: Build output is located in apps/<PROJECT_NAME>/build/build_out/ directory.
