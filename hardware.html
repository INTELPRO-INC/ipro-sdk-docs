IPRO7 晶片概覽

⚠ Release SDK 注意事項：此為 Release SDK v1.0.1。IPRO7 不支援 WiFi。部分進階功能（Matter、Thread、Zigbee、AI/NPU）未包含在此版本中。

2.4G 低功耗無線晶片

IPRO7 是專為低功耗 IoT 應用設計的無線晶片，支援 BLE 5.4 與 LE Audio， 並內建 ISP 影像處理器與 0.1 TOPS NPU 加速器，適用於穿戴裝置、音訊產品與物聯網裝置。

BLE 5.4 LE Audio ISP NPU 0.1 TOPS

MCU 處理器

  • RISC-V RV32IMAFC @192MHz
  • L1 I/D Cache 16KB/16KB
  • RAM 300KB
  • ROM 128KB
  • eFuse 2Kb
  • RTC, Timer×2, WDT
  • DMA 8 通道

記憶體配置

  • 內建 8Mb (1MB) Flash
  • 內建 64Mb (8MB) pSRAM
  • XIP QSPI 支援
  • On-The-Fly AES 解密 (OTFAD)
  • 支援外部 QSPI Flash

無線功能

  • Bluetooth 5.4 / BLE
  • LE Audio (LC3 Codec)
  • 最多 16 個 BLE 連線
  • 2.4 GHz RF

安全功能

  • Secure Boot 安全啟動
  • Secure Debug 安全除錯
  • TrustZone 隔離保護
  • AES-CBC/CCM/GCM/XTS
  • MD5, SHA-1/224/256/384/512
  • TRNG 真隨機數產生器
  • PKA (RSA/ECC 加速)

AI / NPU

  • NPU 運算力 0.1 TOPS
  • AI 偵測/辨識應用
  • 邊緣運算加速
  • 影像/語音 AI 處理

多媒體

  • Audio ADC 24-bit
  • Video MJPEG Codec
  • Camera DVP/SPI 介面
  • Display DBI/SPI 介面
  • USB 2.0

UART Pin Assignment (Function = 7)

UART0 GPIO20/GPIO21 不可更改 — Boot ROM 使用此 UART 進行 firmware download (DL mode)，更改將導致無法透過 UART 燒錄韌體。

UART 使用 兩段式 mux：(1) GPIO 設為 function 7 產生 UART_SIG_N，(2) 透過 GLB_UART_Fun_Sel() 將 signal 映射到 UART0/UART1 的 TX/RX/CTS/RTS。任意 GPIO 皆可作為 UART。

UART 配置範例

/* 使用 GPIO14 作為 UART1_TXD, GPIO15 作為 UART1_RXD */
GLB_GPIO_Cfg_Type uart_pins[] = {
    { GLB_GPIO_PIN_14, GPIO_FUN_UART, GPIO_MODE_AF, GPIO_PULL_UP, 0, 1 },
    { GLB_GPIO_PIN_15, GPIO_FUN_UART, GPIO_MODE_AF, GPIO_PULL_UP, 0, 1 },
};
GLB_GPIO_Init(&uart_pins[0]);
GLB_GPIO_Init(&uart_pins[1]);

/* GPIO14 -> UART_SIG_6, GPIO15 -> UART_SIG_7 */
GLB_UART_Fun_Sel(GLB_UART_SIG_6, GLB_UART_SIG_FUN_UART1_TXD);
GLB_UART_Fun_Sel(GLB_UART_SIG_7, GLB_UART_SIG_FUN_UART1_RXD);

SPI 配置範例

/* SPI0 on GPIO12~15 group */
GLB_GPIO_Cfg_Type spi_pins[] = {
    { GLB_GPIO_PIN_12, GPIO_FUN_SPI, GPIO_MODE_AF, GPIO_PULL_NONE, 1, 1 }, /* MOSI */
    { GLB_GPIO_PIN_13, GPIO_FUN_SPI, GPIO_MODE_AF, GPIO_PULL_NONE, 1, 1 }, /* MISO */
    { GLB_GPIO_PIN_14, GPIO_FUN_SPI, GPIO_MODE_AF, GPIO_PULL_UP,   1, 1 }, /* SS   */
    { GLB_GPIO_PIN_15, GPIO_FUN_SPI, GPIO_MODE_AF, GPIO_PULL_NONE, 1, 1 }, /* SCLK */
};
for (int i = 0; i < 4; i++) GLB_GPIO_Init(&spi_pins[i]);

GPIO28~33 為 SDC 與 SF2/SF3 共用 — 定義 CONFIG_DISABLE_SDC_PINMUX 時，這些 pin 改為 external flash 用途，不可同時使用 SD card。

Pin 衝突與設計注意事項

  • GPIO20/21：Boot ROM UART download 固定使用，不可重新分配
  • GPIO0~2, GPIO11：預設 JTAG Debug，正式產品可釋放使用
  • GPIO17~22：與內部 Flash (SF) 共用，若內部 Flash 啟用則不可使用
  • GPIO23~28：與外部 Flash (SF2) 共用，需注意衝突
  • GPIO28~33：SDC 與 SF2/SF3 互斥，由 CONFIG_DISABLE_SDC_PINMUX 選擇

外設連接指南

I2S 音訊 Codec

連接外部音訊編解碼器 (如 ES8388、WM8978)。使用 I2S Group 1 (GPIO4~7) + I2C0 控制。

IPRO7 Function Codec
GPIO4 (Fun=3, I2S0_BCLK) I2S Bit Clock BCLK
GPIO5 (Fun=3, I2S0_FS) I2S Frame Sync LRCLK / WS
GPIO6 (Fun=3, I2S0_DO) I2S Data Out DIN (to codec)
GPIO7 (Fun=3, I2S0_DI) I2S Data In DOUT (from codec)
GPIO12 (Fun=6, I2C0_SCL) I2C Clock SCL
GPIO13 (Fun=6, I2C0_SDA) I2C Data SDA

I2S 也可使用其他 Group (0~7)，每組 4 pin 依序為 BCLK/FS/DO/DI，詳見 I2S Pin Assignment。

SPI 外設 (LCD / Sensor)

SPI0 接 LCD 或感測器。使用 Group 3 (GPIO12~15)。

IPRO7 Function 外設
GPIO15 (Fun=4, SPI0_SCLK) SPI Clock SCK
GPIO12 (Fun=4, SPI0_MOSI) SPI Master Out SDI / MOSI
GPIO13 (Fun=4, SPI0_MISO) SPI Master In SDO / MISO
GPIO14 (Fun=4, SPI0_SS) Chip Select CS#

MOSI/MISO 可透過 pin_swap_enable 交換。SPI0 支援 8 組 pin group，詳見 SPI Pin Assignment。

SD Card (SDIO)

SD 卡使用固定 GPIO28~33，不可更改。

IPRO7 SD Card
GPIO28 (Fun=27, SDC_CLK) CLK
GPIO29 (Fun=27, SDC_CMD) CMD
GPIO30 (Fun=27, SDC_DAT0) DAT0
GPIO31 (Fun=27, SDC_DAT1) DAT1
GPIO32 (Fun=27, SDC_DAT2) DAT2
GPIO33 (Fun=27, SDC_DAT3) DAT3

GPIO28~33 與外部 Flash (SF2/SF3) 共用。定義 CONFIG_DISABLE_SDC_PINMUX 切換為 Flash 模式。詳見 SDIO Pin Assignment。

I2C 感測器

I2C0 介面感測器連接範例。I2C0 可用任意偶數/奇數 GPIO 配對 (SCL=偶數位, SDA=奇數位)。

IPRO7 Function Sensor
GPIO4 (Fun=6, I2C0_SCL) I2C Clock SCL
GPIO5 (Fun=6, I2C0_SDA) I2C Data SDA
GPIO10 (Fun=11, GPIO) Interrupt INT (optional)
3V3 Power VDD
GND Ground GND

I2C 需要外部 4.7 kOhm 上拉電阻（內部 pull-up ~50 kOhm 不足以驅動標準 I2C bus）。I2C1 (Fun=25) 為獨立的第二組 I2C。

UART 擴展

UART1 可分配到任意 GPIO。範例使用 GPIO14/15。

IPRO7 Function 外設
GPIO14 (Fun=7, UART_SIG6) UART1_TXD RXD
GPIO15 (Fun=7, UART_SIG7) UART1_RXD TXD

UART0 (GPIO20 TXD, GPIO21 RXD) 為 Boot ROM 固定使用，不可更改。詳見 UART Pin Assignment。

2.4GHz RF 天線建議

  • PCB 天線: 成本低，需留出淨空區

設計建議

輸入保護

12V 輸入加裝 TVS (SMAJ15CA) 靜電/浪湧保護

去耦電容

每個電源腳位放置 100nF 去耦電容，靠近 IC

大容量電容

+5V 與 +3.3V 放置 10μF~47μF 電容穩定電源

電源序列

確保 0.9V 核心電源優先建立，再上 IO 電源

接地規劃

使用完整接地面，類比/數位區域分離

LDO 選型

低雜訊 LDO 用於類比電源 (AVDD)
