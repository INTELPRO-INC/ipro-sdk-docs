IPRO7 晶片概覽

💻

處理器核心

  • RISC-V RV64IMAFDC
  • 最高 320MHz 時脈
  • 32KB I-Cache
  • 32KB D-Cache
  • FPU 浮點運算單元
💾

記憶體配置

  • 4MB 內建 Flash
  • 16MB PSRAM
  • 256KB OCRAM
  • 4KB HBNRAM (休眠保持)
  • 支援外部 QSPI Flash
📶

無線功能

  • BLE 5.4 完整支援
  • LE Audio / LC3 編解碼
  • TX Power: -20 ~ +10 dBm
  • RX Sensitivity: -97 dBm
  • IEEE 802.15.4 (可選)
⚡

電源特性

  • 工作電壓: 1.8V ~ 3.6V
  • Active: 約 5mA @ 3.3V
  • Deep Sleep: < 5μA
  • Hibernate: < 1μA
  • 內建 DC-DC 轉換器

⚠️ 注意事項

  • GPIO14/15: 預設為 SWD Debug 介面，燒錄後可重新配置為 GPIO
  • GPIO16: 開機時讀取此腳位，LOW = 進入 Bootloader，HIGH = 正常開機
  • ADC: 輸入電壓範圍 0 ~ 3.3V，請勿超過 VDD
  • RF: 天線匹配電路請參考 Reference Design

天線設計

2.4GHz RF 天線建議

  • PCB 天線: 成本低，需留出淨空區
  • Chip 天線: 體積小，需匹配電路
  • 外接天線: 效能最佳，需 U.FL 連接器

π 型匹配電路:

設計建議

去耦電容

每個電源腳位放置 100nF 去耦電容，靠近 IC

大容量電容

VBAT 放置 10μF 電容，穩定瞬間電流

接地規劃

使用完整接地面，RF 區域獨立接地

電源序列

確保 VDD_CORE 先於 VDD_IO 上電

典型電流消耗

工作模式 典型電流 條件
Active (CPU) 15 mA 160MHz, 執行程式
BLE TX 8.5 mA 0 dBm, 1M PHY
BLE RX 7.5 mA 1M PHY
Light Sleep 50 μA RAM 保持
Deep Sleep 5 μA RTC 運作
Hibernate 0.8 μA 僅保持 HBNRAM
